
1. Authentication
    1.1 Registration
        url: **/api/auth/register
        method: POST
        body: {
                  "name": String,
                  "cnic": String(length 13),
                  "phoneNumber": String(length 11),
                  "email":String(pattern email),
                  "password":String,
                  "userType": String(Capital ) = [DONOR, RECIPIENT, VOLUNTEER, ADMIN]
              }

    1.2 verify otp
    url: **/api/auth/verify-otp
            method: POST
            body : {
                "email" : string,
                "otp" : string
            }


    1.3 Sign in
        url: **/api/auth/login
        method: POST
        body: {
                  "username": String,
                  "password":String
              }
    1.4 user profile
        url : localhost:8080/api/user/profile
        method POST
        body :
        {
         "userProfile": {
             "name": "me",

3. Donor APIs
    3.1 Create Donation
        url: **/api/donor/create-donation
        method: POST
        body: {
                   "donorId": Long,
                   "count": Integer,
                   "surplusMaterialId": Long,
                   "description": "String",
                    "pickup_time": "String",
                     "location": {
                             "Latitude": "String",
                              "Longitude": "String"
                                  }
              }
    3.2 Get Donations
        url: **/api/donor/donations/{id: long}
        method: GET
        Response: List of Previous Donations

    3.3 Get Surplus Materials
        url: **/api/donor/surplusMaterialsTypes
        method: GET
        Response: List of Surplus Materials

    3.4 close donation
        url: localhost:8080/api/donor/close_donation/1
        method: Post
        Response: 200

    3.5 list of active donation
          url : localhost:8080/api/donor/active_donations
          method : GET
          Response  : list of all active donation
