Admin End points
    + add Surplus Material
        - url localhost:8080/api/admin/add-surplus
        - method POST
        - body  {
                    "name": <<String: Name of Surplus Materials>>,
                    "description": <<String: Briefs of Surplus Material>>
                }

Donor End points
    + create donation
        - url localhost:8080/api/donor/create-request
        - method POST
        - body  {
                    "donorId": <<String: donor id>>,
                    "count": <<String: count of donation>>,
                    "surplusMaterialId": <<String: SurplusMaterial id>>
                }


Volunteer End points
    + Engage a donation
        - url localhost:8080/api/volunteer/engage-donation
        - method POST
        - body  {
                    "requestId": <<Long: Donation request id>>,
                    "volunteerId": <<Long: Volunteer id>>
                }

    + Donate a Donation
        - url localhost:8080/api/volunteer/donate-donation
        - method POST
        - body  {
                    "requestId": <<Long: Donation request id>>,
                    "volunteerId": <<Long: Volunteer id>>,
                    "recipientId": <<Long: Recipient id>>
                }


Recipient End point
    + View history of donations
        - url localhost:8080/api/recipient/donation-history
        - method POST
        - body <<long: Recipient ID>>
        - response  {
                        status: <<status of the request>>
                        body: <<List: list of donation previously made>>
                    }

    + Latest donations
            - url localhost:8080/api/recipient/donations
            - method GET
            - response  {
                            status: <<status of the request>>
                            body: <<List: list of donations made to recipient>>
                        }
